# EPIC-03: Upload de screenshots

## Description

Upload et gestion des screenshots de l'application. Détection automatique du type de device et prévisualisation.

## Valeur métier

Permet aux utilisateurs de fournir les visuels de leur app de manière simple et intuitive, base pour la génération des mockups.

## Stories

- [STORY-3.1](./STORY-3.1.md) - Créer la zone de drag & drop
- [STORY-3.2](./STORY-3.2.md) - Implémenter la détection de device
- [STORY-3.3](./STORY-3.3.md) - Gérer la liste et le réordonnancement

## Critères d'acceptance (Epic)

- [ ] L'utilisateur peut drag & drop des images
- [ ] L'utilisateur peut cliquer pour uploader
- [ ] Le type de device est détecté automatiquement
- [ ] Les images sont prévisualisées en thumbnails
- [ ] L'utilisateur peut réordonner les images
- [ ] L'utilisateur peut supprimer une image
