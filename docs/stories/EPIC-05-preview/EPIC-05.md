# EPIC-05: Preview

## Description

Prévisualisation du rendu final simulant la page App Store, avec carousel des mockups et vue des textes.

## Valeur métier

Permet à l'utilisateur de voir exactement ce que donnera sa page App Store avant d'exporter, réduisant les allers-retours.

## Stories

- [STORY-5.1](./STORY-5.1.md) - Créer le composant de preview App Store
- [STORY-5.2](./STORY-5.2.md) - Implémenter le carousel de mockups
- [STORY-5.3](./STORY-5.3.md) - Ajouter le mode plein écran

## Critères d'acceptance (Epic)

- [ ] Preview ressemble à une page App Store
- [ ] Carousel de mockups navigable
- [ ] Textes affichés de manière réaliste
- [ ] Mode plein écran disponible
- [ ] Toggle iPhone/iPad view

## Dépendances

- EPIC-02 (Textes générés)
- EPIC-04 (Mockups générés)
