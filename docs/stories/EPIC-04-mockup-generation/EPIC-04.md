# EPIC-04: Génération de mockups

## Description

Génération de mockups professionnels à partir des screenshots uploadés, avec différents templates et options de personnalisation.

## Valeur métier

Deuxième feature principale. Permet de créer des visuels professionnels sans compétences en design, économisant temps et argent.

## Stories

- [STORY-4.1](./STORY-4.1.md) - Créer les templates de device frames
- [STORY-4.2](./STORY-4.2.md) - Implémenter le renderer de mockups
- [STORY-4.3](./STORY-4.3.md) - Créer le sélecteur de templates
- [STORY-4.4](./STORY-4.4.md) - Ajouter les options de personnalisation
- [STORY-4.5](./STORY-4.5.md) - Implémenter le template text-overlay

## Critères d'acceptance (Epic)

- [ ] 3 templates disponibles : frame, gradient, text-overlay
- [ ] Mockups générés en < 2 secondes
- [ ] Options de personnalisation (couleur device, background)
- [ ] Preview temps réel
- [ ] Résolution App Store correcte

## Dépendances

- EPIC-01 (Setup)
- EPIC-03 (Upload screenshots)
