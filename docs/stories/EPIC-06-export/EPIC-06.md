# EPIC-06: Export

## Description

Export de tous les assets générés (textes + mockups) dans un fichier ZIP prêt pour App Store Connect.

## Valeur métier

Point final du workflow. Permet à l'utilisateur de récupérer tous ses assets en un seul téléchargement, prêts à être uploadés sur App Store Connect.

## Stories

- [STORY-6.1](./STORY-6.1.md) - Créer la modal d'export
- [STORY-6.2](./STORY-6.2.md) - Implémenter la génération du ZIP
- [STORY-6.3](./STORY-6.3.md) - Ajouter les options d'export

## Critères d'acceptance (Epic)

- [ ] Export en 1 clic
- [ ] ZIP contient textes (txt, json) + mockups (png)
- [ ] Structure de dossiers organisée
- [ ] Options pour choisir formats/devices
- [ ] Progression visible pendant génération
